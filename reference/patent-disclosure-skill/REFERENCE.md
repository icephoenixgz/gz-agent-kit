# patent-disclosure-skill (参考项目)
> ⭐ 3,700+ | github.com/handsomestWei/patent-disclosure-skill
> 类型：专利挖掘 & 交底书生成工具（SKILL.md 格式，参考收录）

通用中国专利挖掘发现与交底书生成全流程：扫描项目文档挖掘专利点、讨论融合、基于脱敏模版生成技术交底书、联网查新、生成后自检含逻辑闭环与公式参数一致性。

## 流程
1. 边界与输入 → 2. 项目文档扫描 → 3-4. 专利挖掘与融合 → 5. 联网查新 → 6. 摘要预览 → 7. 交底书构建 → 8. 自检

## 亮点
- 完整迭代模式（merger / correction_handler 双模式）
- 国知局查新脚本（cnipa_epub_search.py）
- mermaid 渲染工具
- 13个 prompt 文件，覆盖全流程

## 在 gz-agent-kit 中的定位
作为研究/专利板块的参考工具。若涉及医工交叉或医疗器械方向的专利，可作为基础框架。
