## What this project does not promise

- It does not guarantee bypassing AI detectors.
- It does not rewrite every text aggressively.
- It does not replace human editorial judgment.
- It does not copy rules from other humanizer projects; it acknowledges them and implements its own compact rule set.
- It has not yet been fully tested in Claude Code or OpenCode.
